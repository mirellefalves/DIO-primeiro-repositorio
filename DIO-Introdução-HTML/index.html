Definição

HTML é uma Linguagem de Marcação de Hipertexto comumente utilizada na construção de páginas Web.

Pode ser interpretada por navegadores e tem como sintaxe básica a que se nota na imagem a seguir.

Formatação do Texto

No HTLM existem algumas tags específicas para exibir o texto com uma formatação específica.

  • b - texto em negrito
  • strong - texto importante
  • i - Texto em itálico
  • em - Texto em destaque
  • mark - Texto marcado
  • small - Texto menor
  • del - Texto excluído
  • ins - Texto inserido
    • sup - Texto sobrescrito
  • sub - Texto subscrito
